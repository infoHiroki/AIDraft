# AIDraft システム納品書

## 納品日
2025年6月

## 納品内容

### 1. 納品方法

#### 🔧 開発環境
- **デプロイツール**: Google Clasp

#### 📦 納品手順
⚠️ **重要**: アカウント情報の一時共有が必要（業界標準手法）

1. **クライアントのGoogleアカウント情報を一時的に共有**
   - 既存スプレッドシートアクセスのため必須
   - 作業完了後にパスワード変更で対応

2. **開発者による代理作業**
   ```bash
   # クライアントアカウントでログイン
   clasp login
   
   # 各プロジェクトディレクトリで実行
   cd 01_お問い合わせ && clasp create --title "AIDraft-お問い合わせ" && clasp push
   cd 02_単発 && clasp create --title "AIDraft-単発" && clasp push  
   cd 03_WeekendEnt && clasp create --title "AIDraft-WeekendEnt" && clasp push
   ```

3. **クライアント環境での設定・確認**
   - OpenAI APIキー設定
   - 権限許可
   - 動作確認テスト
   - 設定完了状態で引き渡し

4. **セキュリティ対応**
   - 開発者側でログアウト完了
   - クライアントによるパスワード変更

### 2. Google Apps Script プロジェクト（3システム）

#### 📋 01_お問い合わせ
**機能**: GDX歯科セミナー問い合わせ自動返信システム  
**対象シート**: お問い合わせ  
**処理列**: F列（質問）→ G列（ステータス）+ H列（統合情報）

#### 📋 02_単発  
**機能**: 単発セミナーアンケート質問自動回答システム  
**対象シート**: 単発  
**処理列**: H列（質問）→ I列（ステータス）+ J列（統合情報）

#### 📋 03_WeekendEnt
**機能**: WeekendEntセミナーアンケート質問自動回答システム  
**対象シート**: WeekendEnt  
**処理列**: G列（質問）→ H列（ステータス）+ I列（統合情報）

### 3. システム仕様
- **AI API**: OpenAI GPT-4o-mini
- **処理対象**: Google Sheetsの各種データ
- **出力**: Gmail下書き自動作成
- **管理**: スプレッドシートでの処理状況記録
- **アーキテクチャ**: 独立したGASプロジェクト × 3

### 4. 主要機能
- 質問内容の自動分析・返信生成
- Gmail下書きの自動作成
- 処理状況のリアルタイム更新
- エラーハンドリング・ログ管理
- 統合出力形式（日時・Draft ID・AI回答）

### 5. 提供ドキュメント
- ✅ **AIDraft運用ガイド** - 日常運用マニュアル
- ✅ **納品書** - 契約内容・サポート範囲
- ✅ **列対応仕様書** - データ処理仕様

## 設定要件

### 必須設定（各プロジェクト共通）
1. **OpenAI APIキー**の設定
   - Google Apps Script スクリプトプロパティに登録
   - キー名: `OPENAI_API_KEY`

2. **Gmail権限**の許可
   - 初回実行時に権限付与が必要

3. **スプレッドシート権限**の確認
   - 対象シートへのアクセス権限
